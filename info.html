Examples

Red and Green blink alternating with a 4 sec period (each 2 sec on/2 sec off)

{ "red":[2,2], "green":[2,2], "delay":2 }

Red blinking, 2 sec on 1 sec off

{ "red":[2,1] }

Example using httpie

http PUT hostname red:='["10","10"]' green:='["10","10"]' delay=10

Or,

just add /0, /1, /2, /3 to the URL to set: 0: both off, 1: green on, 2: red on, 3: both on
